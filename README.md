# 🛡 SmartSpam AI

AI Powered Spam Detection System using Machine Learning.

## Features

- Spam Detection
- Confidence Score
- Risk Level Analysis
- URL Detection
- Spam Category Classification
- Dashboard Analytics
- Modern Dark UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-Learn
- Pandas

## Machine Learning Model

- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Accuracy

96.68%

```bash
pip install -r requirements.txt
python app.py
```
